Stop trackbacks after linking a blog post with some other internal blog post.



== Description ==

Stop receiving pingbacks when linking a post to your own blog post.

Documentation - http://wp.me/p1trMn-1ZM

== Installation ==

1. Upload plugin's .zip to `/wp-content/plugins/` directory
2. Activate the plugin through 'Plugins' menu in WordPress
